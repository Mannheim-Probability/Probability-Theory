Exercise 1 (Arzelà-Ascoli).

We are going to prove the Arzelà-Ascoli theorem in this exercise. While it is usually formulated with (relative) compactness, we have already seen this is equivalent to total boundedness in a complete space.

Definition (Equicontinuity).

Let (E,d),(F,d) be metric spaces and ℱ a set of functions from E to F. Then ℱ is equicontinuous iff

∀ε>0,∃δ>0,∀f∈ℱ:d⁢(x,y)<δ⟹d⁢(f⁢(x),f⁢(y))<ε.

In other words: δ is independent of f∈ℱ (and also of x,y, i.e. we also have uniform continuity).

  1. (i)

    Prove that any finite subset of C⁢(K) for a compact space K is equicontinuous.

    Solution.

    Let {f1,…,fd}⊆C⁢(K). As any continuous function is uniformly continuous on a compact set, we have

    ∀ϵ>0,∀i∈{1,…,d},∃δi>0:d⁢(x,y)<δi⟹d⁢(fi⁢(x),fi⁢(y))<ϵ

    With δ:=mini=1,…,d⁡δi we get equicontinuity. ∎

  2. (ii)

    Prove that total boundedness of a subset of (C(K),∥⋅∥∞) implies equicontinuity.

    Solution.

    Let ℱ⊆C⁢(K) be a totally bounded set and choose some ϵ>0. Then we have by total boundedness f1,…,fd such that for all f∈ℱ exists an fi with

    ∥f-fi∥∞<ϵ4.

    And since the f1,…,fd are equicontinuous by (i), we can find some δ>0, such that

    d⁢(x,y)<δ⟹|fi⁢(x)-fi⁢(y)|<ϵ2 ∀i=1,…,d

    Putting things together we have for all f∈ℱ, all x,y∈K with d⁢(x,y)<δ

    |f⁢(x)-f⁢(y)| ≤|f⁢(x)-fi⁢(x)|⏟≤∥f-fi∥∞⁣≤ϵ4+|fi⁢(x)-fi⁢(y)|⏟≤ϵ2+|fi⁢(y)-f⁢(y)|⏟≤∥f-fi∥∞⁣≤ϵ4
    ≤ϵ.∎
  3. (iii)

    Let ℱ⊆C⁢(K) be a pointwise bounded set, i.e.

    ∀x∈K,∃M⁢(x)>0:|f⁢(x)|<M⁢(x) ∀f∈ℱ.

    Prove that for any finite set {x1,…,xd}⊆K, the set

    Φ={(f⁢(x1),…,f⁢(xd)):f∈ℱ}

    is totally bounded.

    Solution.

    Φ is bounded in (ℝd,∥⋅∥∞) by maxi=1d⁡M⁢(xi). Therefore it is totally bounded as this is equivalent on ℝd. ∎

  4. (iv)

    Prove that pointwise boundedness and equicontinuity of ℱ⊆C⁢(K) imply total boundedness (with regard to the sup-norm ∥⋅∥∞).

    Hint.

    Use equicontinuity to extend (iii) to all points. Do not forget that K is totally bounded!

    Solution.

    For a given ϵ>0 we want to find an ϵ covering. Since ℱ is equicontinuous, we can select δ>0, such that for all d⁢(x,y)<δ we have

    |f⁢(x)-f⁢(y)|≤ϵ4 ∀f∈ℱ. (1)

    Since K is totally bounded, we can find a δ covering, i.e. there exist x1,…,xd, such that for all x∈K there exists k with

    d⁢(x,xk)≤δ. (2)

    Due to total boundedness of Φ by (iii) there exist f1,…,fm such that for any ϕ⁢(f)∈Φ we can find i with

    ∥ϕ⁢(f)-(fi⁢(x1),…,fi⁢(xd))⏟=⁣:ϕ⁢(fi)∥∞<ϵ2. (3)

    So take any f∈ℱ and choose fi such that (3) is satisfied. Then for all x∈K there exists an xk by (2) such that d⁢(x,xk)<δ and hence

    |f⁢(x)-fi⁢(x)|≤|f⁢(x)-f⁢(xk)|⏟≤(⁢1⁢)ϵ4+|f⁢(xk)-fi⁢(xk)|⏟≤∥ϕ⁢(f)-ϕ⁢(fi)∥∞⁣≤ϵ2+|fi⁢(xk)-fi⁢(x)|⏟≤(⁢1⁢)ϵ4

    As x was arbitrary we have

    ∥f-fi∥∞≤ϵ.

    Therefore f1,…,fm are an ϵ covering of ℱ, proving its total boundedness. ∎

Putting (ii) and (iv) together we therefore have

Theorem (Arzelà-Ascoli).

Let K⊆E be totally bounded in a metric space (E,d). A set of continuous functions mapping to ℝ, ℱ⊆C⁢(K), is totally bounded with regard to the sup-norm iff it is pointwise bounded and equicontinuous.

  1. (v)

    Deduce that 𝒫:={x↦xn:n∈ℕ}⊆C⁢([0,1]) is bounded but not totally bounded.

    Solution.

    Since p⁢(x)∈[0,1] for all p∈𝒫, the set is bounded. To show that it is not totally bounded, we will show it is not equicontinuous. Take any ϵ>0 and choose some δ>0. Then there exists a polynomial such that

    |p⁢(1)-p⁢(1-δ)|=1-(1-δ)n>ϵ

    for some n∈ℕ. Therefore the set 𝒫 can not be equicontinuous. ∎
