Exercise 5.

Let 𝒵≔{t≥0:Bt=0} be the zero set of a Brownian motion. Show that, almost surely,

  1. (i)

    𝒵 is closed

  2. (ii)

    𝒵 is unbounded

  3. (iii)

    𝒵 has zero Lebesgue measure

Solution.
  1. (i)

    For almost-every ω∈Ω, t⁢ ↦Bt⁢(ω) is continuous so {t≥0:Bt⁢(ω)=0} is closed.

  2. (ii)

    For almost-every ω∈Ω, lim supt→∞⁡Bt⁢(ω)=∞, lim inft→∞⁡Bt⁢(ω)=-∞ and t⁢ ↦Bt⁢(ω) is continuous so {t≥0:Bt⁢(ω)=0} is unbounded.

  3. (iii)

    Let λ⁢(𝒵) be the Lebesgue measure of 𝒵. Using Fubini’s Theorem, we have

    𝔼⁢[λ⁢(𝒵)] =∫Ω(∫0∞𝟙Bt⁢(ω)=0⁢dt)⁢dℙ⁢(ω)
    =∫0∞(∫Ω𝟙Bt⁢(ω)=0⁢dℙ⁢(ω))⁢dt
    =∫0∞ℙ(Bt=0)dt.

    Now for every t>0, Bt is a Gaussian random variable so that ℙ(Bt=0)=0 and hence 𝔼⁢[λ⁢(𝒵)]=0.

  4. (iv)

    Remark first that since, almost surely, for every ε>0,

    supt∈[0,ε]⁡Bt>0 and inft∈[0,ε]⁡Bt<0,

    then by continuity, almost surely, for every ε>0, there exists t∈(0,ε) such that Bt=0. As a consequence, inf⁡{t>0:Bt=0}=0 almost surely.

    For every q∈ℚ+, we consider the stopping time dq≔inf⁡{t≥q:Bt=0}∈𝒵. It is almost surely finite by (ii) and, using strong Markov property, the process Bq defined by

    Btq≔(Bdq+t-Bdq)⁢𝟙dq<∞⁢=(d)⁢Bdq+t⁢𝟙dq<∞

    is a standard Brownian motion. Then

    ℙ(inf{t>dq:Bt=0}=dq)=ℙ(inf{t>0:Bdq+t=0}=0)=ℙ(inf{t>0:Btq=0}=0)=1.

    Hence, almost surely, dq is not isolated for every q∈Q+.

    For ω∈Ω, consider T⁢(ω)∈{t≥0:Bt⁢(ω)=0} and a sequence of rational (rn)n≥1 such that rn↑T⁢(ω) as n→∞. Then for every n≥1, rn≤drn⁢(ω)≤T⁢(ω), which implies that drn⁢(ω)→T⁢(ω) as n→∞. Thus T⁢(ω) is not isolated and almost surely, 𝒵 has no isolated point.
