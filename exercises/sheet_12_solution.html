Exercise 3.

Let X be an integrable random variable. Set X+=X∨0 and X-=-X∨0, such that X=X+-X-.

  1. (i)

    Show that 𝔼⁢[X-|X+]=-𝔼⁢[X⁢𝟙X≤0]ℙ(X≤0)⁢𝟙X+=0.
    Hint: by using the factorization lemma, we may try to find a function g such that
    𝔼⁢[X-|X+]=g⁢(X+).

  2. (ii)

    Determine 𝔼⁢[X|X+]

Solution.

Let g be an integrable function [0,∞)→ℝ such that g⁢(X+)=𝔼⁢[X-|X+]. Then for any integrable function f:[0,∞)→ℝ, by definition of conditional expectation there is the identity

𝔼⁢[g⁢(X+)⁢f⁢(X+)]=𝔼⁢[X-⁢f⁢(X+)].

Notice that, we have

𝔼⁢[X-⁢f⁢(X+)]=𝔼⁢[0⁢f⁢(X)⁢𝟙X>0]+𝔼⁢[-X⁢f⁢(0)⁢𝟙X≤0]=-f⁢(0)⁢𝔼⁢[X⁢𝟙X≤0]

and

𝔼[g(X+)f(X+)]=𝔼[g(X+)f(X+)𝟙X>0]+𝔼[g(0)f(0)𝟙X≤0]=𝔼[g(X+)f(X+)𝟙X>0]+g(0)f(0)ℙ(X≤0).

Combining these, we have the identity

-f(0)𝔼[X𝟙X≤0]=𝔼[g(X+)f(X+)𝟙X>0]+g(0)f(0)ℙ(X≤0). (1)

As this holds for every integrable function f:[0,∞)→ℝ, in particular, when f⁢(x)=𝟙x=0, we deduce that

-𝔼[X𝟙X≤0]=g(0)ℙ(X≤0)⟹g(0)=-𝔼⁢[X⁢𝟙X≤0]ℙ(X≤0).

It follows from the observation and (1) that for every integrable function f:[0,∞)→ℝ,

0=𝔼⁢[g⁢(X+)⁢𝟙X+>0⁢f⁢(X+)].

Therefore, g⁢(X+)⁢𝟙X+>0=0 ℙ-a.s. and thus

g⁢(X+)=g⁢(X+)⁢𝟙X+=0=g⁢(0)⁢𝟙X+=0=-𝔼⁢[X⁢𝟙X≤0]ℙ(X≤0)⁢𝟙X+=0.

This completes the proof of (i).

For (ii), we observe that

𝔼⁢[X|X+]=𝔼⁢[X+|X+]-𝔼⁢[X-|X+]=X++𝔼⁢[X⁢𝟙X≤0]ℙ(X≤0)⁢𝟙X+=0.

∎

Exercise 4 (Symmetric random walk).

Let (Xn)n≥1 be a sequence of i.i.d. random variables with ℙ(X1=1)=ℙ(X1=-1)=1/2. Let a,b∈ℕ with 0<a<b. Define S0≔a and for every n≥1, Sn≔Sn-1+Xn. Finally, define the following stopping time:

T0≔inf⁡{n≥0:Sn=0}⁢and⁢Tb≔inf⁡{n≥0:Sn=b}.

Let T=T0∧Tb.

  1. (i)

    Deduce the values of ℙ(T0<Tb) and ℙ(T0>Tb).

  2. (ii)

    Compute the value of 𝔼⁢[T].

  3. (iii)

    Show that ℙ(T0<∞)=1 and ℙ(Tb<∞)=1.

  4. (iv)

    I have 20 euros initially and I play in a fair game. I have 2 different choices: either I always bet for 10 euros every round; or I always bet for 1 euro each round. I will leave the game whenever I have 100 euros or I lose all my money. Which one should I choose, in order to reduce my risk to lose all?

Hint: you have many martingales for use. Be careful with the assumptions in optional stopping theorem.

Solution.
  1. (i)

    Consider the martingale Sn∧T: it is bounded, hence uniformly integrable. So Sn∧T converges to a limit in L1 and a.s.. So it stays at a point after a long enough time. The limit cannot be any of the interior point. So it has to be either 0 or b. This observation implies that T<∞ a.s..

    As a consequence, we have limn→∞⁡Sn∧T=ST a.s.. By dominated convergence and martingale property:

    a=𝔼[S0]=limn→∞𝔼[Sn∧T]=𝔼[ST]=0⋅ℙ(T0<Tb)+bℙ(T0>Tb). (2)

    So we have ℙ(T0>Tb)=a/b and ℙ(T0<Tb)=(b-a)/b.

  2. (ii)

    Using the martingale Mn:=Sn2-n: we have

    a2=𝔼⁢[M0]=𝔼⁢[Mn∧T]=𝔼⁢[Sn∧T2-n∧T].

    By dominated convergence:

    limn→∞⁡𝔼⁢[Sn∧T2]=𝔼⁢[ST2].

    By monotone convergence:

    limn→∞⁡𝔼⁢[n∧T]=𝔼⁢[T].

    Then we deduce from (2) that

    a2=𝔼[ST2]-𝔼[T]=b2ℙ(T0>Tb)-𝔼[T]=ab-𝔼[T].

    So 𝔼⁢[T]=a⁢(b-a).

  3. (iii)

    Let b→∞:

    ℙ(T0=∞)≤lim supb→∞ℙ(T0>Tb)=limb→∞a/b=0.

    By symmetry, we also have ℙ(Tb=∞)=0.

  4. (iv)

    If it is a fair game, the risk to win is ℙ(T0>Tb)=a/b. With a=20,b=100 or with a=2,b=10, the probability is the same.

∎

Exercise 5.

Let (Pn) be a sequence of probability measures on E′×E′′. Show that, (Pn) is tight if and only if the two sets of marginal distributions are tight are E′ and E′′.

Solution.

The projection map ϕ1:E′×E′′→E′ defined by ϕ1⁢(x,y)=x is continuous. Therefore, for any compact set K⊂E′×E′′, the projection ϕ1⁢(K) is a compact set in E′′. Conversely, for K′⊂E′,K′′⊂E′′ compact sets, K′×K′′ is a compact set. Using these facts, we easily derive the statement from the definition of tightness. ∎

Exercise 6.

Let (Bt,t≥0) be a standard Brownian motion.

  1. (i)

    For an arbitrary t>0, calculate 𝔼⁢[Bt4] and 𝔼⁢[|Bt|].

  2. (ii)

    For an arbitrary t>0, and a∈ℝ, calculate 𝔼⁢[exp⁡(a⁢Bt)].

  3. (iii)

    Let ℱ1:=σ(Bs,s∈[0,1]). Calculate 𝔼⁢[B5|ℱ1] and 𝔼⁢[B52|ℱ1].

  4. (iv)

    Show that 𝔼⁢[|∫[0,1]Bs/s⁢𝑑s|]<∞.

  5. (v)

    Let βt:=Bt-∫[0,t]Bss⁢𝑑s. Show that (βt,t≥0) is a Brownian motion.

Solution.
  1. (i)

    𝔼⁢[Bt4]=3⁢t2, 𝔼⁢[|Bt|]=2⁢t/π.

  2. (ii)

    𝔼⁢[ea⁢Bt]=e12⁢a2⁢t. A few more useful formula: 𝔼⁢[Bt⁢ea⁢Bt]=a⁢t⁢e12⁢a2⁢t, and when a<12⁢t, 𝔼⁢[ea⁢Bt2]=11-2⁢a⁢t

  3. (iii)

    By independent increments:

    𝔼⁢[B5|ℱ1]=𝔼⁢[B1+B5-B1|ℱ1]=B1+𝔼⁢[B5-B1|ℱ1]=B1

    and

    𝔼⁢[B52|ℱ1] = 𝔼⁢[B12+2⁢(B5-B1)⁢B1+(B5-B1)2|ℱ1]
    = B12+4
  4. (iv)

    Because 𝔼⁢[|Bs|]=2π⁢s, we have by Fubini’s theorem:

    ∫01𝔼⁢[|Bs/s|]⁢𝑑s=∫012π⁢s-1/2⁢𝑑s<∞

    By Fubini’s theorem, we have:

    𝔼⁢[|∫01Bs/s⁢𝑑s|]≤𝔼⁢[∫01|Bs/s|⁢𝑑s]=∫01𝔼⁢[|Bs/s|]⁢𝑑s<∞
  5. (v)

    From (v), we have a.s. ∫0tBs/s⁢𝑑s<∞, thus βt is well defined. We see that B0=0, and βt is a Gaussian process: ∀ai∈ℝ,1≤i≤n, the linear combination

    ∑i=1nai⁢βti=∑i=1nai⁢Bti-∑i=1nai⁢∫0tiBs/s⁢𝑑s

    is the limit of a sequence of Gaussian variables, thus it is a Gaussian variable itself. It follows that for ∀t1<t2<⋯<tn, the vector (βt1,⋯,βtn) is a Gaussian vector, thus βt is a Gaussian process.
    Also from the proof of (v), we have 𝔼⁢[∫01Bs/s⁢𝑑s]=∫01𝔼⁢[Bs/s]⁢𝑑s=0, thus the process βt is centered.
    Next, for ∀s≤t, the covariance C⁢o⁢v⁢(βs,βt):

    C⁢o⁢v⁢(βs,βt) = 𝔼⁢[(Bs-∫0sBu/u⁢𝑑u)⁢(Bt-∫0tBu/u⁢𝑑u)]
    = 𝔼⁢[Bs⁢Bt]-𝔼⁢[∫0sBt⁢Bu/u⁢𝑑u]-𝔼⁢[∫0tBs⁢Bu/u⁢𝑑u]+𝔼⁢[∫0sBu/u⁢𝑑u⁢∫0tBu/u⁢𝑑u]

    Because ∫0t𝔼⁢[|Bs⁢Bu|]/u⁢𝑑u<∞, ∫[0,t]2𝔼⁢[|Bs⁢Bu|]/(u⁢v)⁢𝑑u⁢𝑑v<∞, we can use Fubini’s theorem:

    C⁢o⁢v⁢(βs,βt)
    = s-∫0s𝔼⁢[Bt⁢Bu]/u⁢𝑑u-∫0t𝔼⁢[Bs⁢Bu]/u⁢𝑑u+∫0s∫0t𝔼⁢[Bu⁢Bv]/(u⁢v)⁢𝑑u⁢𝑑v
    = s-∫0su/u⁢𝑑u-∫0t(s∧u)/u⁢𝑑u+∫0s∫0t(u∧v)/(u⁢v)⁢𝑑u⁢𝑑v
    = s-s-(s+s⁢ln⁡(t/s))+(s⁢ln⁡(t/s))=s

    And finally, we see the continuity of path from the continuity of path for Bt.

∎
