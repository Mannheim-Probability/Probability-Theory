Exercise 1 (Borel σ-Algebra).

Let (E,d) be a metric space with Borel σ-Algebra

ℬ(E):=σ({O⊆E:O open)}
  1. (i)

    Assume there exists a countable base π of the topology τ of E. Prove that σ⁢(π)=ℬ⁢(E).

    Solution.

    For any set O∈τ we have O=⋃nOn with On∈π. This implies O∈σ⁢(π) and therefore τ⊆σ⁢(π). But since π⊆τ we have

    ℬ⁢(E) =σ⁢(τ)=σ⁢(π).∎
  2. (ii)

    Assume that E is separable (has a countable dense subset A). Prove that

    π:={Bq⁢(x):x∈A,q∈ℚ}

    is a (countable) base of τ.

    Remark.

    In case of ℝ, this base π is the set of all intervals with rational endpoints.

    Hint.

    ⋃{B∈π:B⊆O}=O

    Solution.

    Let O∈τ. We claim that

    U:=⋃{B∈π:B⊆O}=O.

    By definition U⊆O holds. So consider y∈O. Since O is open, there exists an ϵ>0 (without loss of generality in ℚ) such that Bϵ⁢(y)⊆O. Since A is dense, there exists x∈A with d⁢(x,y)<ϵ2 (otherwise Bϵ2⁢(y)⊆A¯c=Ec=∅). This implies

    y∈Bϵ2⁢(x)⏟∈π⊆Bϵ⁢(y)⊆O.

    Since y was arbitrary we have U⊇O. ∎

  3. (iii)

    Prove that σ({Bϵ(x),x∈E,ϵ>0})=ℬ(E) if E is separable.

    Solution.

    Using π from (ii), we have

    ℬ⁢(E) =(i)σ(π)⊆σ({Bϵ(x),x∈E,ϵ>0})⊆ℬ(E).∎
