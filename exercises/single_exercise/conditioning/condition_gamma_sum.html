Exercise 1 (Conditional Expectation).

Recall that a gamma distribution with parameter c>0 and θ>0 has density:

θcΓ⁢(c)⁢xc-1⁢e-θ⁢x⁢𝟙x>0.
  1. 1.

    Let X,Y be two independent exponential random variables with parameter θ>0 and Z=X+Y. Determine the conditional distribution of X given Z=z (i.e. the Markov kernel (A,z)↦ℙ(X∈A|Z=z)).

    Solution.

    For all g non-negative and measurable functions we have that:

    𝔼⁢[h⁢(X)⁢g⁢(Z)] =𝔼⁢[h⁢(X)⁢g⁢(X+Y)]
    =∫ℝ2h⁢(x)⁢g⁢(x+y)⁢fX⁢(x)⁢fY⁢(y)⁢dx⁢dy
    =∫ℝ2h⁢(x)⁢g⁢(x+y)⁢θ⁢exp⁡(-θ⁢x)⁢𝟙{x≥0}⁢θ⁢exp⁡(-θ⁢y)⁢𝟙{y≥0}⁢dx⁢dy
    =∫ℝ2h⁢(x)⁢g⁢(x+y)⁢θ2⁢exp⁡(-θ⁢(x+y))⁢𝟙{x≥0}⁢𝟙{y≥0}⁢dx⁢dy
    =z=x+y∫ℝ2h(x)g(z)θ2exp(-θz))𝟙{x≥0}⁢𝟙{z-x≥0}⏟=𝟙z≥0⁢𝟙z≥x≥0dxdy
    =∫ℝg⁢(z)⁢θ2⁢e-θ⁢z⁢𝟙{z≥0}⁢z⁢(1z⁢∫0zh⁢(x)⁢dx)⁢dz
    =𝔼⁢[g⁢(Z)⁢(1Z⁢∫0Zh⁢(u)⁢du)]

    Since the last term is ℱZ-measurable we can conclude that 𝔼⁢[h⁢(X)|Z]=∫0Z1Z⁢h⁢(u)⁢du. Using ℙ(X∈A|Z)=𝔼[𝟙A(X)|Z] means that for A∈ℬ⁢(ℝ), we get

    ℙ(X∈A|Z=z)=∫𝟙A1z𝟙[0,z]du=λ⁢([0,z]∩A)z.

    Hence, the conditional distribution of X given Z=z is 𝒰⁢[0,z]. ∎

  2. 2.

    Conversely, let Z be a random variable with gamma distribution with parameter (2,θ), and suppose X is a random variable whose conditional distribution given Z=z is uniform on [0,z], for z>0. Prove that X and Z-X are independent with exponential distribution Exp⁢(θ).

    Solution.

    With kernels we can easily write

    𝔼⁢[h⁢(X,Z)|Z] =Thm. 5.3.5∫ℝh(u,Z)ℙ(X∈du|Z) (1)
    =∫ℝh⁢(u,Z)⁢1Z⁢𝟙{0≤u≤Z}⁢du (2)

    where (A,z)↦ℙ(X∈A|z) is the regular conditional distribution defined from the unique kernel we get via the disintegration theorem. Essentially doing all the same calculations backwards we get

    𝔼⁢[f⁢(X)⁢g⁢(Z-X)] =𝔼⁢[𝔼⁢[f⁢(X)⁢g⁢(Z-X)|Z]]
    =𝔼⁢[∫ℝf⁢(u)⁢g⁢(Z-u)⁢1Z⁢𝟙{0≤u≤Z}⁢du]
    =∫ℝ2f⁢(u)⁢g⁢(z-u)⁢1z⁢𝟙{0≤u≤z}⁢θ⁢z⁢e-θ⁢z⁢𝟙z≥0⁢du⁢dz
    =∫ℝ2f⁢(u)⁢g⁢(y)⁢θ2⁢e-θ⁢(u+y)⁢𝟙y≥0⁢𝟙u≥0⁢du⁢dy
    =∫ℝf⁢(u)⁢θ⁢e-θ⁢u⁢𝟙u≥0⁢du⁢∫ℝg⁢(y)⁢θ⁢e-θ⁢y⁢𝟙y≥0⁢dy.

    This shows that (X,Z-X) are independent Exp⁢(θ) random variables. In more detail: Use g≡1 to show X∼Exp⁢(θ). And similarly f≡1 to show Z-X∼Exp⁢(θ). Independence follows using indicator functions in place of g,f. ∎
