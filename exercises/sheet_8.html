Exercise 1.

Let (Ω,ℱ,μ) be a measure space, f,g:Ω→ℂ an integrable function. Show that

  1. (i)

    ∫Ωa⁢f+g⁢d⁢μ=a⁢∫Ωf⁢dμ+∫Ωg⁢dμ, for all a∈ℂ

  2. (ii)

    Re⁢(∫Ωf⁢dμ)=∫ΩRe⁢(f)⁢dμ, Im⁢(∫Ωf⁢dμ)=∫ΩIm⁢(f)⁢dμ

  3. (iii)

    ∫Ωf⁢dμ¯=∫Ωf¯⁢dμ

Exercise 2.
  1. (i)

Exercise 4.

For any k≥1, a random variable Z:=(Z1,…,Zk) on ℝk is called a centred Gaussian vector, if every linear combination of Z1,…,Zk is a centred Gaussian random variable on ℝ.

  1. (i)

    Prove that, Z is a centred Gaussian vector, if and only if

    𝔼⁢[exp⁡(i⁢t⋅Z)]=exp⁡(-12⁢tT⁢M⁢t),∀t∈ℝk,

    where M is a k×k matrix, with Mi⁢j=Cov⁢(Zi,Zj).

    Hint.

    recall that the characteristic function of 𝒩⁢(0,σ2) is φ⁢(t)=exp⁡(-12⁢σ2⁢t2)

  2. (ii)

    Let (X1,…,Xm,Y1,…,Yn) be a centred Gaussian vector. Prove that, (X1,…,Xm) is independent of (Y1,…,Yn), if and only if Cov⁢(Xi,Yj)=0 for all i≤m,j≤n.
